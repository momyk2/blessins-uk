Images placed here are NOT included in `assets/portfolio-manifest.json` (they are skipped by
`scripts/generate-portfolio-manifest.mjs`).

Use this folder for:
- Nearly blank / solid-black frames you do not want in the hero tunnel
- One-off exports you still want on disk but not in the live grid

After moving files, run from project root:
  node scripts/generate-portfolio-manifest.mjs
