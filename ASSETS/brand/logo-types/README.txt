Logo Design slide — one image per tile (for independent animation)

Drop PNG or WebP files here using these exact names (lowercase):

  wordmark.png          — text / script style mark
  lettermark.png        — initials / monogram
  brandmark.png         — symbol-only mark
  combination-mark.png — icon + word together
  emblem.png            — badge / seal style
  mascot.png            — character mark
  abstract.png          — abstract mark
  dynamic.png           — responsive / lockup variants (one frame is fine)

Background for the whole slide stays separate:
  ../wordmark.png   (or replace with your own wide photo — same path in index.html if you rename)

If a file is missing, the tile still shows the label; add the image when ready.
